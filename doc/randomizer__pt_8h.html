int perf_test ( )

Function shows result of performance check on stdout.

Parameters
[out]statusSuccess(0)/Fail(1)

First creates random numbers and saves it in array for performance test. Then calls algorithms with input array in random, asc., then desc. order. For asc. order the result of sorting of random numbers is used. Result of asc. order is reversed to serve as input for sorting with desc. order.
