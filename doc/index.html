Search and sort algorithms

Overview

This program written in C implements well known sort and search algorithms. The following algorithms are implemented:

  1. Bubblesort
  2. Insertionsort
  3. Mergesort
  4. Quicksort

Special features

  1. Menu
  2. Colored output on stdout.
  3. Each sorting algorithm can be run on its own.
